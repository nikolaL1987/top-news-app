## React news app

Building a React app which makes use of https://newsapi.org/ API.

## Introduction

Simple React App takes top news Data from NewsAPI to keep you updated with latest news.

## App details

News API https://newsapi.org/ to get the news articles from different sources.

### `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br>
You will also see any lint errors in the console.
